RAW的檔案格式

基於TIFF格式

  1. 標準header(metadata)
  2. 拍攝資訊(EXIF)
  3. 電子資訊
  4. 縮圖

Nikon Electronic Format (NEF)

  1. 白平衡
  2. 曝光度
  3. 色彩轉換
  4. 動態高光
  5. 鏡頭資訊
