基本指令

Command Purpose Example
ls list content ls ./
cd change directory cd ../
cp copy file cp a.cpp b.cpp
mv move file(rename) mv a.cpp b.cpp
rm remove file(dangerous) rm a.cpp
mkdir create directory mkdir tmp

進階指令

Command Purpose Example
su root login su
sudo do as root sudo apt-get update
exit logout exit
chmod change privilege chmod 777 a.cpp
echo display echo 1
cat display file cat a.cpp

控制指令

Command Purpose Example
; 執行兩個指令 ls ; pwd
&& 執行第二個當第一個成功 ls && pwd
|| 執行第二個當第一個失敗 ls || pwd
> 把執行結果寫入檔案(複寫) ls > a.txt
>> 把執行結果寫入檔案(加入) ls >> a.txt
< 讀入檔案當作stdin輸入 a.cpp < test.txt >
