# 駭客

對電腦科學、程式和設計方面具高度理解的人

--

# 黑帽駭客

通常指怪客(Cracker)，以利益導向  
專門尋找系統或程式漏洞，入侵網站、竊取個人資料  
牟取不當的利益等為他們的目標｡

--

# 白帽駭客

資訊安全人員，具有駭客的知識與能力，了解駭客的手法  
扮演正義的一方，能夠修補網站漏洞及程式漏洞，阻絕黑帽駭客的攻擊｡
